HTML


HTML

is the standard language for creating web pages. HTML describes the structure of a web page using markup. HTML elements are the building blocks of HTML pages. 

        
        <!DOCTYPE html>
        <html>
          <head>
            <title>Page Titl</title>
          </head>
          <body>
            <h1>This is a Heading</h1>
            <p>This is a paragraph.</p>
          </body>
        </html>

The 

<mark>
tag highlights text, while 
<del>
and 
<ins>
tags show deleted and inserted text, respectively.
